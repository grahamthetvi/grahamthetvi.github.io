CVI Type Talker

Type letters to hear them spoken aloud.

Press Space or Enter to hear the whole word and see a picture.

Press Backspace to delete a letter.

Press ← → arrow keys to flip through pictures of the same word.

Press Ctrl+Shift+M to show words-per-minute speed.

Press Ctrl+Shift+L to show letters-per-minute speed.

Press Ctrl+Shift+C to clear the screen.

⌨️ This app requires a physical keyboard. It will not work on a touchscreen without one connected.

⚙️ Settings Guide

Here's a quick overview of everything you can customise:

🔤 Text Appearance

Change the font style, size, and colour of the typing display, plus the background colour.

🔵 Bubble Letters

Add a coloured outline around every letter to make text pop against any background.

⏱ Typing Control

Set a minimum delay between key presses — helpful for students who press keys accidentally or too quickly.

🖼 Image Display

Control how large the picture appears. Pre-load images for specific words so they show instantly when typed.

🚫 Image Filtering

Block certain words from ever showing a picture, or restrict the app to an approved word list only.

← → Image Navigation Arrows

Show or hide the on-screen arrow buttons for flipping through multiple photos. The keyboard arrow keys always work too.

🖱 Mouse Cursor

Change the cursor style — useful if a student is using a mouse to click the navigation arrows.

📋 Session Word History

A running log of every word typed this session with timestamps — useful for teachers reviewing progress.
